## üí° KEY FEATURES AT A GLANCE

### Mode Selection
```
User selects: "Slab" or "Level"
System saves mode to database
UI shows colored badge (Blue=Slab, Green=Level)
Commission calculated using selected mode
```

### Compensation Breakdown
```
Gross Sales: Total sales before returns
Returns: Amount deducted (shown in red)
Net Sales: Calculated gross - returns (shown in green)
Commission: Based on net sales + mode
Total Payout: Base salary + commission
```

### Full Transparency
```
Users can see:

‚úÖ How much was returned (returns)
```
